Cull Pilot

Rate, sort, and curate your photos with purpose—faster.

Built for Synology Photos

Cull Pilot is a browser extension that adds fast, keyboard-driven controls to the Synology Photos web interface. Apply star ratings, navigate images, and curate your collection—all without touching the mouse.

Efficient by Design

Inspired by Vim, Cull Pilot's keyboard controls are optimized for speed and muscle memory. Press i to enter insert mode, then use 1–5 to rate, h and l to move between images, and Esc to exit.

For Pros and Newcomers Alike

If you're already familiar with star ratings in tools like Lightroom, Cull Pilot gives you a faster way to apply them directly in the Synology Photos web app. Your existing workflow stays intact—you just move faster.

If you're new to the concept: Synology Photos lets you build albums using star rating criteria—like “4 stars and up” or “rated within the last month.” With Cull Pilot, you can tag those images on the fly while reviewing them.

What It Does

  • Star images using keys 0 through 5
  • Clear a rating by pressing the same number twice or using 0
  • Navigate between photos with h and l
  • Lightweight overlay shows active mode and rating feedback
  • Works entirely in lightbox view—no clicks required

Open Source and Community-Supported

Cull Pilot is open source and built for speed, clarity, and trust. You can inspect the code, suggest features, or contribute directly via GitHub.
