Cull Pilot

Rate, sort, and curate your photos with purpose—faster.

Built for Synology Photos

Cull Pilot is a browser extension that adds fast, keyboard-driven controls to the Synology Photos web interface. Star images, navigate albums, and curate selections without ever touching the mouse.

Efficient by Design

Cull Pilot is inspired by Vim—its keyboard controls are designed for speed and muscle memory. Press i to enter insert mode, then use 1–5 to rate, h and l to navigate, and Esc to exit.

For Pros and Newcomers Alike

If you're already familiar with star ratings from tools like Lightroom or photo metadata (EXIF), Cull Pilot gives you a faster way to apply those ratings directly from the Synology web app. Your existing workflows stay intact—you’re just moving faster.

If you're new to the concept: star ratings in Synology Photos can be used to automatically build albums based on criteria like “4 stars and up” or “rated within the last month.” With Cull Pilot, you can build these albums quickly by starring images directly while reviewing them.

What It Does

  • Star images using keys 0 through 5
  • Clear a rating by pressing the same number twice or using 0
  • Navigate between photos using h and l
  • Overlay shows active mode and feedback without interfering
  • Works entirely in the lightbox view—no need to click the info panel

Open Source and Transparent

Cull Pilot is open source and built for clarity, speed, and trust. You can inspect the code, suggest features, or contribute directly via GitHub.
